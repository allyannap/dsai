# Using Python Virtual Environment

## Setup (one-time)
1. Navigate to this directory: `cd /Users/allyanna/Desktop/Coding/sysen5381/dsai/01_query_api`
2. Create virtual environment: `python -m venv .venv`
3. Activate it: `source .venv/bin/activate`
   - Your prompt should show `(.venv)` at the start
4. Install requests: `pip install requests`

## Running scripts (every time you open a new terminal)
1. Navigate to this directory: `cd /Users/allyanna/Desktop/Coding/sysen5381/dsai/01_query_api`
2. Activate the virtual environment: `source .venv/bin/activate`
3. Run your script: `python 02_example.py`

## Deactivating
When done, type: `deactivate`
